# value-adapters

LoRA Adapters fine-tuning and routing for Multicultural tasks.


## Set up environment
```bash
conda create -n valadapt python=3.10
conda activate valadapt
pip install -r requirements.txt
```

## Running Experiments
Run the script in the "script" folder, pick a particular experiment that you need to run.
The script and the environment are conditioned to be fixed.
